# nfc-card-reader-react-native

Read credit cards using NFC

## Installation

```sh
npm install nfc-card-reader-react-native
```

## Usage


```js
import { multiply } from 'nfc-card-reader-react-native';

// ...

const result = multiply(3, 7);
```
